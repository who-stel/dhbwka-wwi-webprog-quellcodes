Programmiere hier das Layout aus dem Screenshot nach. Das Endergebnis muss
nicht exakt der Vorlage entsprechen. Das Layout sowie die wesentlichen
Gestaltungselemente (Hintergründe, Farben, Schriftgrößen usw.) sollten aber
einigermaßen übereinstimmen.

Als Hintergrundbild für das „Jumbotron” (großer bunter Bereich oben mit dem
Eye Catcher) kannst du das Bild jumbotron-hintergrund.png verwenden. Die
Farbgebung ist ähnlich wie das Original. Allerdings handelt es sich um ein
frei verfügbares Bild, dessen Verwendung in dieser Aufgabe keine Urheberrechte
verletzt. :-)

https://pixabay.com/illustrations/geometric-wallpaper-abstract-3037028/

CSS-Anweisungen für Farbverläufe können z.B. auf https://cssgradient.io
erstellt werden. Hiermit wurde auch folgender Farbverlauf erstellt, welcher
der Vorlage recht nahe kommen sollte:

background: rgb(221,145,17);
background: linear-gradient(90deg, rgba(221,145,17,1) 0%, rgba(255,9,28,1) 45%, rgba(250,0,255,1) 100%);
