Das Cicada 3301 Mysterium

Cicada 3301 ist der Name, mit dem eine unbekannte Organisation, die seit 2012 mehrfach komplexe Rätselserien veröffentlichte, in den Medien benannt wird.
Der Name wird auch für die Rätselwettbewerbe selbst verwendet. ( Wikipedia )
